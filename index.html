Explanation of widget data parameters

data-id:
OpenEBench : search your tool and use the last part of url of the found tool as data-id e.g https://dev-openebench.bsc.es/html/ws/#!/tool/ trimAl
bio.tools : search your tool and use the last part of url of the found tool as data-id e.g https://bio.tools/ trimAl
data-widget-type:
"widget": default widget
data-widget-subtypes (add several according to your needs):
"widget": widget container
"title": tool name
"bottom_badge": bottom badge
"badge": badge only
data-widget-size: "225" (as minimum) is recommeded
class="opeb": all widget containers have this class associated
